## 🧭 About This Repository

This repository is a comprehensive, self-paced journey through **Data Structures and Algorithms** using **Java**. It covers everything from fundamental programming constructs (loops, methods, recursion) to advanced algorithmic techniques (backtracking, greedy algorithms, divide & conquer).

Every solution is organized by topic into its own folder, making it easy to study one concept at a time or use as a quick reference during interview preparation.

> 🎯 **Goal:** Build a strong algorithmic foundation by solving structured, categorized problems — one topic at a time.

## 🔍 Problem Categories & Techniques

### Arrays (70 problems)
A deep dive across every major array paradigm:

- **Two Pointer:** Two Sum, Container With Most Water, Merge Sorted Array
- **Sliding Window:** Max Average Subarray, Max Sum of Distinct Subarray, Longest Substring Without Repeating Chars
- **Binary Search:** Binary Search, Search in Rotated Sorted Array, Find First & Last Position, Find Peak Element
- **Prefix/Suffix:** Product of Array Except Self, Subarray Sum Equals K, Find Prefix Common Array
- **Matrix:** Spiral Matrix I & II, Set Zeros, Search in 2D
- **Intervals:** Merge Intervals, Trapping Rain Water
- **Sorting-based:** Sort Colors, Inversion Count, Partition Array
- **Math / Bit tricks:** Single Number, Missing Number, First Missing Positive, Duplicate Number

### Strings (25 problems)
- Palindrome detection & partitioning
- Anagram validation, string compression
- Roman ↔ Integer conversion
- ZigZag conversion, String-to-Integer (atoi)
- Add Binary, permutation checks

### Backtracking (11 problems)
- Classic constraint satisfaction: N-Queens (both variants)
- Combinatorics: Permutations, Subsets I & II
- Grid exploration: Grid Ways, Knight's Tour
- String problems: Palindrome Partitioning, Subset of String

### Greedy (12 problems)
- Scheduling: Activity Selection, Find Longest Chain
- Optimization: Fractional Knapsack, Array Partition
- Circuit: Gas Station
- Array: Destroying Asteroids, Split Array

### Recursion (16 problems)
- Classic series: Fibonacci, Factorial
- Search: First/Last/All occurrences in array
- Counting: Climb Stairs, Friends Pairing, Tiling Problem
- Utility: Tower of Hanoi, Reverse using recursion

### Stack (12 problems)
- Monotonic Stack: Next Greater Element, Next Smallest Element, Max Area of Histogram
- Expression: Polish Notation, Valid Parentheses
- Design: Min Stack
- Path: Simplify Path
